How to Use

Calculator
To use the calculator;
  1. Enter a number in the first input area labelled "first number"
  2. Select an operator from the dropdown list
  3. Enter a number in the second input area labelled "second number"
  4. Click the "answer" button for result
  5. Click the "reset" button to clean the fields
Converter
To use the converter, (Note: It only converts currency to/from NGN and USD)
  1. Enter a number in one of the input area labelled "1st quantity" or "2nd quantity"
  2. Click the "answer" button for result
  3. Click the "reset" button to clean the fields
Matrix Calculator
To use the matrix calculator,
  1. Enter a number in one of the input area labelled "Number of Registered Accounts"
  2. Click the "answer" button for result
  3. Click the "reset" button to clean the fields
